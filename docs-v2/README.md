# docs-v2 (DEPRECATED)

This directory contains an archived Astro-based documentation site.
It is **no longer actively maintained**. 

The current documentation lives in the `docs/` directory using Jekyll.

## Why is this here?

This was an experimental migration to Astro/Starlight that was not adopted.
The files are kept for reference purposes only.

## Removal

If you want to remove this directory entirely, delete it or run:
```bash
git rm -r docs-v2/
```
