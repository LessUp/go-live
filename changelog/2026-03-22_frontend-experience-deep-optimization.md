# 前端体验深度优化

日期：2026-03-22

## 变更内容

- 新增共享前端静态资源层，统一首页、推流、播放、录制页的样式与基础交互
- 新增 `GET /api/bootstrap`，向浏览器下发 ICE、鉴权状态与页面功能开关
- 推流页与播放页统一 WebRTC 生命周期、停止操作、状态提示与日志输出
- 录制列表页改为安全 DOM 渲染，补充加载态、空态与错误态
- README 与中文 README 补充控制台入口和 bootstrap 接口说明

## 背景

原有示例页脚本和样式全部内联，前后端 ICE 配置不一致，推流/播放生命周期分散，录制页使用 `innerHTML` 渲染服务端数据，难以继续扩展且存在体验与安全隐患。本次优化保持 Go 内嵌静态多页架构不变，在当前结构上补齐共享运行时和浏览器端配置入口。
